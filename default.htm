Instructions

  1. Create an html document.
  2. Add a div tag in the body of the document.
  3. Add script tags referencing the following scripts.
    1. http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.6.2.min.js
    2. http://ajax.aspnetcdn.com/ajax/jquery.ui/1.8.15/jquery-ui.min.js
    3. arcGisJsApi.js
  4. Add a script tag containing the following line of code. (In the example below, your div element has an id of map. $("#map").arcGisMap();

    This will create a map object on element with an id of map.
